# Digital-Notes-Project
# Digital Notes Management System

This repository contains documentation for the Digital Notes Management System project. It includes:
- Presentation slides
- Project guidelines
- Software Requirements Specification (SRS)
- Software Design Document (SDD)

## Objective
The project aims to develop an efficient system for managing digital notes, allowing users to create, organize, categorize, and collaborate on notes.

## Files:
- Presentation.pdf: The project presentation.
- Guidelines.pdf: The project guidelines document.
- SRS.pdf: Software Requirements Specification.
- SDD.pdf: Software Design Document.
